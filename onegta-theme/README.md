# OneGTA WordPress Theme v2.0

Полноценный фан-портал серии Grand Theft Auto.

## Структура файлов

```
onegta-theme/
├── style.css                    # Главный CSS + заголовок темы
├── functions.php                # Весь PHP-функционал
├── header.php                   # Шапка + навигация + auth modal
├── footer.php                   # Подвал + newsletter
├── index.php                    # Главная страница
├── archive.php                  # Архив (новости/статьи/файлы/видео)
├── single.php                   # Одиночный пост/новость/статья
├── single-videos.php            # Страница видео
├── single-files.php             # Страница файла
├── 404.php                      # 404
├── page-profile.php             # Профиль пользователя (Template)
├── page-submit.php              # Добавить материал (Template)
├── page-cheats.php              # Читы (Template)
├── inc/
│   └── helpers.php              # Хелперы (breadcrumb, embed)
├── js/
│   └── main.js                  # Весь JavaScript
├── template-parts/
│   └── cards/
│       ├── post-card.php        # Карточка поста/новости/статьи
│       ├── file-card.php        # Карточка файла
│       └── video-card.php       # Карточка видео
└── images/                      # Папка для изображений темы
```

## Установка

### 1. Загрузка темы
- Распакуй `onegta-theme.zip` в `/wp-content/themes/`
- Активируй в **Внешний вид → Темы**

### 2. После активации
- Перейди в **Настройки → Постоянные ссылки** → выбери "Название записи" → Сохрани
- Это активирует все CPT и ЧПУ

### 3. Создай страницы

Создай следующие страницы в **Записи → Страницы**:

| Название         | URL (ярлык) | Шаблон                    |
|------------------|-------------|---------------------------|
| Профиль          | profile     | Профиль пользователя      |
| Добавить материал| submit      | Добавить материал         |
| Читы GTA         | cheats      | Читы GTA                  |

### 4. Меню
- **Внешний вид → Меню**
- Создай меню "Основное" и назначь на позицию "Основное меню"
- Добавь пункты: Новости, Статьи, Файлы, Видео, Читы
- Для кнопки "GTA VI" добавь CSS-класс `menu-item-cta` (включи "CSS-классы" в Настройках экрана)

### 5. Виджеты
- **Внешний вид → Виджеты**
- Доступные области: Сайдбар — Статьи, Сайдбар — Новости, Сайдбар — Файлы

## Фронтенд функционал

### Регистрация / Вход
- Модальное окно в хедере
- Кнопки "Войти" и "Регистрация" — без редиректа в wp-admin
- После входа — редирект на `/profile/`

### Профиль пользователя (`/profile/`)
- Аватар (через Gravatar), имя, bio, дата регистрации
- Статистика: кол-во материалов по типам
- Вкладки: Материалы / Настройки / На модерации
- Смена имени, email, пароля, bio

### Добавление материалов (`/submit/`)
Доступно только авторизованным. Типы:
- 📰 **Новость** — заголовок, раздел, обложка, текст, теги
- 📖 **Статья** — + выбор игры, тип (гайд/прохождение/читы), сложность
- 📦 **Файл** — загрузка файла drag&drop + внешняя ссылка, категория, игра, версия
- 🎬 **Видео** — YouTube/Rutube ссылка, категория, описание

**Права публикации:**
- Администраторы и редакторы — публикуют сразу
- Обычные пользователи — материал уходит на модерацию (`pending`)

### Файловый архив
- Счётчик скачиваний (AJAX)
- Прямая загрузка файла через кнопку "Скачать"
- Drag&drop загрузка на странице submit

## Часто задаваемые вопросы

**Q: Пользователи не могут добавлять файлы?**
A: Убедись что в WordPress → Настройки → Обсуждение разрешена регистрация пользователей.

**Q: Как разрешить пользователям публиковать без модерации?**
A: В панели администратора выдай роль "Автор" нужным пользователям.

**Q: Как изменить список читов?**
A: Открой `page-cheats.php`, найди массив `$cheats` и редактируй.

**Q: Аватары не работают?**
A: Аватары берутся с Gravatar. Пользователь должен зарегистрировать свой email на gravatar.com.
